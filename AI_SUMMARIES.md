### [Link](https://powerbi.microsoft.com/en-us/blog/drillthrough-in-excel-now-supported-for-direct-lake-and-directquery-models/)
Here are four technical bullet points summarizing the Power BI blog for a professional:

* Power BI now supports drillthrough for Direct Lake and DirectQuery semantic models via Excel's "Show Details" feature, allowing users to retrieve underlying detail rows by double-clicking a PivotTable cell.
* The updated functionality utilizes the MDX DRILLTHROUGH statement, which was previously only supported for Import models, and now enables consistent drillthrough experiences across Import, Direct Lake, and DirectQuery models.
* Drillthrough respects applied filters and security roles, including Row-Level Security (RLS) and Object-Level Security (OLS) configurations, ensuring that users can only access authorized data.
* The performance of drillthrough depends on the underlying source system and query complexity, particularly for semantic models in DirectQuery mode connecting to external data sources, and can be optimized by using explicit measures or fields that support detail rows.
